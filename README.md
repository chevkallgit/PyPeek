# PyPeek
VS Code Extension that alows you to peek into variables to see Metadata and values from last execute

## The Problem
Python can get messy when you cant see exactly what is happening behind dynamic vaiables. This project is for those people who need help traseing data and knowing exactly what is going where.

## The Solution
The aim of this project is to build an extesion for VS Code that can alow us to look inside a vaiable without being inside of a debugger. The UI will be a simple hover of any variable to show the whole field of meta data or the values of the lastest exicute.

## Features
- Show Metadat of variables
- Show latest exicuted values of variables
- Install from VS Code Extensions
